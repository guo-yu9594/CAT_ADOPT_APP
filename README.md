Dependance back:
```bash
cd back
npm install express
npm install request
npm install cors
```

Dependance front:
```bash
cd front
npm install axios
npm i @mui/material
```

To start front (port: 3000):
```bash
cd front
npm start
```

To start back (port: 3001):
```bash
cd back
DEBUG=myapp:* npm start
```
